Robotics Playlist

This page is for robotics projects, especially work related to TritonDroids. Right now, my main robotics project is the MiniAloha robot.

Track List

TritonDroids MiniAloha Robot

Problem: MiniAloha is a small robot project under TritonDroids. The goal is to get the robot's hardware and software working together so it can move reliably and support future features like SLAM and robot arms.

Contribution: I set up the Raspberry Pi, helped configure the robot, and got the motors working. At this stage, the base robot is able to run, while SLAM and the robot arms are being handled by another team member.

Tools: Raspberry Pi, motor setup, robotics hardware, software control, debugging, testing, team communication, and documentation.

Constraints: Debugging took patience because issues could come from the Pi setup, motor wiring, configuration, software, or the physical robot itself.

Status: The base robot is working. Remaining work includes SLAM and the robot arms, which are part of the larger team build.

What I learned: Robotics debugging is different from normal software debugging because the problem is not always in the code. I had to check the setup, hardware, and software together before deciding what was wrong.

Playlist Notes

I want to keep building robotics experience because it forces me to think about the whole system, not just one file or one function.
